Fiscal Analysis Lab

This is where the theory comes to life. In this hands-on lab, you will use a Jupyter Notebook to build and interact with your own AI agents to analyze a real-world financial document.

Launch Lab Notebook

Note: The server may take a few moments to start up.

Video Walkthrough

Objective

The goal of this lab is to build a specialized AI agent that can read and answer questions about a PDF of a corporate fiscal report. You will learn how to load the document, process its content for an AI to understand, and then build agents capable of both simple lookups and complex analysis.

Step-by-Step Guide

  1. 1

    Setup and Environment

    Install the necessary software and securely load your AI Refinery credentials.

  2. 2

    Document Processing

    Load the fiscal report PDF, extract its text, and break it into smaller, AI-friendly chunks.

  3. 3

    Create a Custom Search Tool

    Build a specialized search function that allows an agent to find relevant information within the document.

  4. 4

    Build a Research Agent

    Create a simple agent and equip it with your custom search tool to answer specific questions.

  5. 5

    Build a Flow Super Agent

    Set up an advanced agent that can orchestrate a multi-step analysis for more complex queries.
